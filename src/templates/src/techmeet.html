Mark the day 27th August 2018.
After all, knowledge is a rare thing-you gain by giving it away.

Machine Learning

Machine Learning and Artificial Intelligence are often stereotyped as something very complicated, whereas it just Mathematics.
The concepts of AI and ML were discussed, explaining the workflow of various general purpose machine like SIRI, Assistant, etc.
Also, some hot topics like Embedded AI were discussed.
There was also a student project showcase on pose-detection using tensorflow.js and Real-time Object Detection using YOLO.

-Tanuj Vishnoi

Web Development

Under Django web development, we discussed difference between a website & a webapp & how using python based framework, we can build a secure website.
Then there was a talk on well-konwn sites that are running on Django, afterwards, we had a demonstration of our project i.e. MSC official website & ongoing project of Thapar Sports website.

In the discussion of Android app development, time was devoted in talking about scope of android dev in India & upcoming internship opportunities.
Live demonstration of an app built by the team that is ongoing & will soon be released with MAX hospitals.

-Chirag Mahawar & Rahul Raj

Cyber Security

Taught about basics of network, different type of exploits, how to stay safe on the internet.
There was also a live demonstration and detailed explanation of DOS attack.

-Himanshu Bansal

Internet Of Things

Students were taught about
What is iot...
Application of iot....
Brief discription of bolt(platform specially designed for iot),
Project based on iot example trash maker, car-e-ok, plus future application of iot based large scale program.

-Sushmita Srivastava

Cloud & Big Data

Cloud
Taught about the paths which can be taken up for career in cloud computing known as IaaS, SaaS, PaaS

Big Data
Introduce the students to Hadoop and what basically is categorized as big Data and why it is needed for those who want to pursue data science and what resources to refer and how cloud computing helps in it.

-Bhavay Pahuja

Competetive Programming

A session to get to know everything you need to, on Competitive Programming.
Discussion on and answer to : Why Competitive Programming?
What are the things you need to do/know before starting Competitive Programming?
How does participating in contests help?
What all competitions you can participate in and on which platforms?

-Ashish Niranjan

Javascript

The workshop aimed at introducing the students with Web technologies both frontend as well as backend.
An overview of ReactJS along with server side NodeJS frameworks was given, concluding with a basic intro to Open source.

-Tathagat Thapliyal
